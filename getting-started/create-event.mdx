## Step-by-Step Guide

Follow these simple steps to create your first event on PeerPop.

<Steps>
  <Step title="Create Event">
   Access the create event page by:
   - Clicking the **"Create an Event"** tab in the sidebar
   - Clicking the **"Create"** button on the top right of any page
  </Step>
  <Step title="Choose which platform to sell tickets on">
   - Select **"Create an Event on PeerPop"** to sell tickets on PeerPop (recommended)
   - Select **"Import from another platform"** to import your event from an external ticketing platform and follow these steps:
     - Select the platform you want to import from
     - Enter your event's details
     - Click **"Import Event"** 
  </Step>
  <Step title="Add Media">
    Upload your event assets:
    - Cover image (recommended size: 1200x630px)
    - Additional photos (up to 10)
    - Promotional video (optional)
  </Step>
  <Step title="Set Up Tickets">
    Configure your ticket options:
    - Create ticket tiers (GA, VIP, etc.)
    - Set prices and quantities
    - Add early bird or group discounts
  </Step>
  <Step title="Configure Promotion">
    Set up your promotional strategy:
    - Enable affiliate rewards
    - Set promoter commission rates
    - Create promo codes
  </Step>
</Steps>
