## Promoter List

View all promoters for your event. Each entry shows:
- Promoter name
- Join date
- Tickets sold
- Total commission earned

<Tip>
  Click the dropdown arrow next to any promoter to view additional details like username, phone number, and email address.
</Tip>

## Invite Promoters

<Steps>
  <Step title="Send Invitation">
    1. Click "Add Promoter" button
    2. Enter email or username
    3. Set commission rate (optional)
    4. Send invitation
  </Step>
  
  <Step title="Share Link">
    Alternatively, share your promoter signup link:
    - Copy unique event link
    - Share via social media
    - Track signups automatically
  </Step>
</Steps>

## Search & Sort

Find specific promoters quickly:
- Search by name using the search bar
- Sort any column by clicking column headers
- Filter by performance metrics

## Mass Communication

Send messages to selected promoters:
- Select promoters using checkboxes
- Choose "Select All" for entire list
- Send [Mass Text](/manage-events/mass-texts) or [Mass Email](/manage-events/mass-emails)
