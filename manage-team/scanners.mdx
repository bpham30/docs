## Scanner List

View all scanners for your event. Each entry shows:
- Scanner name
- Phone number
- Status
- Tickets scanned

<Tip>
  Click the dropdown arrow next to any scanner to view additional details and scanning statistics.
</Tip>

## Invite Event Scanners

<Steps>
  <Step title="Add Scanner">
    Enter scanner details:
    - Name
    - Phone number
  </Step>
  
  <Step title="Send Invitation">
    Scanner will receive:
    - SMS invitation
    - Event details
    - Scanning instructions
  </Step>
</Steps>

## Eventbrite Integration

<Warning>
  For events imported from Eventbrite:
  - Only reward tickets can be scanned through PeerPop
  - All other Eventbrite tickets must be scanned using the Eventbrite organizer app
</Warning>

## Mass Communication

Send messages to selected scanners:
- Select scanners using checkboxes
- Choose "Select All" for entire list
- Send [Mass Text](/manage-events/mass-texts) or [Mass Email](/manage-events/mass-emails)
