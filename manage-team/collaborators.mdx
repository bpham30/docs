## Collaborator List

View all collaborators for your event. Each entry shows:
- Collaborator name
- Status
- Invited date
- Sales Percentage
- Permissions

<Tip>
  Click the **More Info** icon next to any collaborator to view contact information and edit permissions.
</Tip>

## Invite Collaborators

<Steps>
  <Step title="Add Collaborator">
    Click **Add Collaborator** button
  </Step>
  
  <Step title="Set Permissions">
    Assign access levels:
    - Edit event settings
    - View attendee data
    - View promoter data
    - Hide collaborator from event page
  </Step>
  <Step title="Choose Sales Percentage">
    Set a desired percentage of the sales revenue to be paid to this collaborator
    Ex:
    - 0% = No commission
    - 50% = 50% of revenue
  </Step>
  <Step title="Send Invitation">
    Invite collaborator via:
    - Text
    - Email
    - PeerPop username
  </Step>
</Steps>

## Mass Communication

Send messages to selected collaborators:
- Select collaborators using checkboxes
- Choose "Select All" for entire list
- Send [Mass Text](/manage-events/mass-texts) or [Mass Email](/manage-events/mass-emails)
