## Sending Mass Emails

<Steps>
  <Step title="Select Recipients">
    Choose who receives the email:
    - Select individual recipients with checkboxes
    - Use header checkbox to select all
    - View selected count
  </Step>
  <Step title="Click Mass Emails">
    Click the **Mass Emails** button to compose your message.
  </Step>
  <Step title="Compose Email">
    <AccordionGroup>
      <Accordion title="Content Options">
        - Include event description
        - Add event link
      </Accordion>
      
      <Accordion title="Draft Email">
        Use our rich text editor to format your email:
        - Text styling
        - Images
        - Links
        - Tables
        - Embeds
        - Custom HTML
      </Accordion>
    </AccordionGroup>
  </Step>
  <Step title="Send Emails">
    Click **Send Emails** button to send your message.
  </Step>
</Steps>
