## Event Settings

<Note>
  Changes are saved automatically as you edit. Toggle your event to "Draft" mode if you want to make multiple changes without affecting the live event.
</Note>

## Basic Information

<Steps>
  <Step icon="pen" title="Event Details">
    Update core event information:
    - Event title
    - Description (or use AI generator)
    - Categories & tags
    </Step>

  <Step icon="clock" title="Date & Time">
    Manage event schedule:
    - Start date & time
    - End date & time
    - Timezone
    </Step>

  <Step icon="location-dot" title="Location">
    Set event location:
    - Venue name
    - Address
    - Virtual link (for online events)
  </Step>
  <Step icon="eye" title="Visibility Settings">
    - Public or private event
    - Hide event host name from event page
    - Hide Attendee guest list from event page
    - Hide share button from event page
  </Step>
</Steps>


## Additional Settings

<CardGroup cols={4}>
  <Card title="Billing Address" icon="location-dot">
    Require a billing address from attendees on checkout.
  </Card>
  <Card title="Questionaire" icon="question">
    Add custom questions that attendees will need to answer before checkout.
  </Card>
  
  <Card title="Terms of Service" icon="shield-halved">
    Add a terms of service agreement to your event.
  </Card>

  <Card title="Donations" icon="money-bill">
    Accept donations from attendees.
  </Card>
</CardGroup>

## Delete Event
Clicking the **Delete** button will remove your event from the platform. This action is irreversible and cannot be undone, so please proceed with caution.
