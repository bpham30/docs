## Individual Invites

<Tabs>
  <Tab title="Phone Number">
    <Steps>
      <Step title="Enter Details">
        Required information:
        - First name
        - Last name
        - Phone number
      </Step>
      
      <Step title="Optional Discount Code">
        Configure promoter discount:
        - Select applicable tickets
        - Choose discount type (% or $)
        - Set discount amount
        - Create custom code
      </Step>
    </Steps>
  </Tab>
  
  <Tab title="Username">
    <Steps>
      <Step title="Enter Username">
        - PeerPop username
      </Step>
      
      <Step title="Optional Discount Code">
        Same discount options as phone invite
      </Step>
    </Steps>
  </Tab>
</Tabs>

## Mass Invites

<Tabs>
  <Tab title="Share Link">
    Create shareable promoter link:
    1. Set commission rate (optional)
    2. Include discount code (optional)
    3. Share link to promoters for them to join
  </Tab>
  
  <Tab title="Contact Group">
    Invite existing contact group:
    1. Select contact group
    2. Set commission rate
    3. Include discount code (optional)
    4. Send mass invitation
  </Tab>
  
  <Tab title="CSV Upload">
    Import multiple promoters:
   
    1. Navigate to **Upload Data** in the overall marketing page on the sidebar
    2. Import your promoters via CSV upload
    3. Return back to this page and import **All Promoters** by via mass invite by contact groups
  </Tab>
</Tabs>


## Generated Links

  View all your generated invite links by clicking the **View Promoter Invite Links** dropdown

  Each link shows:
  - Link URL & copy button
  - Commission rate

<Note>
  Once invited, promoters will receive their unique tracking link and appear in your [Promoter List](/manage-team/promoters).
</Note>
